# snake-zig

A vibe coded console Snake game written in Zig using libvaxis for terminal UI.

###

![https://gdonald.github.io/88x31/i/vibe_coded.gif](https://gdonald.github.io/88x31/i/vibe_coded.gif)

## Requirements

- Zig 0.15.2

## Build

```bash
zig build
```

## Run

```bash
zig build run
```

Press `q` or `Ctrl+C` to quit.

## Test

```bash
zig build test
```
